What is CSS and why is it important for the website?

CSS is an abbreviation from Cascade Style Sheets and is an important thing if you want to add style to your website. This language is different from HTML and requires some skills to use. However, it's mandatory if you want to become a qualified markup specialist, frontend or even full-stack developer. So let's see how this all things are happening with CSS and HTML.

What is the structure of a common CSS rule?

In order to work, CSS rules should be properly structured. We will show examples of how it should be done when using CSS in head or in separate file. The following image illustrates the general CSS rule:

As you can see, common CSS rule consists of selector and declaration. Declaration is put in special brackets. It can contain one or several rules. Each rule in its turn consists of property and value separated by semi-column and rules are separated by trailing dot-n-comma.

How does CSS allow to work with fonts?

CSS is a great way to change the fonts properties and to change the overall style of your page in the end. Let us see the most common CSS font properties in the following table with examples:

CSS Font Properties

Property Explanation

font-family

This property allows to change the actual font, e.g. Arial, Trebuchet, etc.

font-size

Allows to change the size of font

text-align

Allows to align text or other elements to left, center or right

text-transform

transforms the text in various ways, making it uppercase, lowercase etc.

How to change page layout with CSS

CSS can be used to change page layout of HTML page. The most common examples are stated below

Page Layout CSS Examples

Property Explanation

margin

Adds a margin for a certain block allowing it to stand out from others

padding

Defines the distance of content from borders of the block

text-align

Allows to align text or other elements to left, center or right

height and width

Allow to define the height and width of block or element
